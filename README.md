## Prerequisite: 
- [Node.js](https://nodejs.org/en/download/).

## :rocket: Run

:bulb: Execute the package passing your email and password:

```sh
npx bbbot email password
```

Example with email `foo@bar.com` and password `123456`:

```sh
npx bbbot foo@bar.com 123456
```
